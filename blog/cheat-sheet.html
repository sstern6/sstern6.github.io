JavaScript objects manipulation and notation

Creating an object

  • Create an empty object:
    • var object = {};
  • Add properties and values to an object:
    • var object = { foo: 1,
          "bar": "some string"
          };

Object syntax

  • Object implicit syntax
    • implicit_form = {"Property 1" => "Value 1",
          }
  • Object alternate syntax
    • alternate_form = { Property_1: "Value 1",
          }

Accessing properties

  • Access first value of object using dot notation:
    • object.foo
          //1

Nested properties

  • Create nested object:
    • var objectNested = { baz:
          { foo:
          { bar: 5
          }
          }
          };
  • Access the 5 value of objectNested:
    • object.baz.foo.bar;
          //5

Key takeaways and reference links

  1. Objects and nested objects properties can be accessed using 2 syntaxes, the most common is the dot notation:
  2. object.foo;
  3. object.baz.foo.bar;
