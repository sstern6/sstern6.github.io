Creating an object

  • Create an empty object:
  • var object = {};
  • Add properties and values to an object:
  • var object = {foo: 1,"bar": "some string",};

Object syntax

  • Object implicit syntax
  • implicit_form = {"Property 1" => "Value 1",}
  • Object alternate syntax
  • alternate_form = { Property_1: "Value 1", }

Accessing properties

  • Access first value of object using dot notation:
  • object.foo => //1

Nested properties

  • Create nested object:
  • var objectNested = { baz: { foo: { bar: 5 } } };
